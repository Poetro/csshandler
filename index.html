What is this?

One of the big problems in sitebuilds: You can't make a good group from css and javascript handlers. You need to handle this on many place.

This script is good solution for this on simple way:

/* Test function for #test_selector1 handle (this all are javascript) */
/*ch
function tester1(selector) {
    $(selector).click(function(){
        alert(selector);
    });
}
ch*/
button#test_selector1 { /*ch tester1('$selector'); */ }

Result:

Use
  1. Include javascript codes:
    Pattern: 
    /*ch
    javascript code what you want to include
ch*/
  2. Run javascript codes:
    Pattern: 
    /*ch javascript_function_without_spaces [repeat [delay_in_ms]] */

    Replacable vars:
    $selector : called from this selector block
    $repeat : called with this repeat number
    $delay : called with this delay time
    $counter : repeat cycle counter
  3. Easy sample with jquery: 
    #test_selector2 { /*ch $('$selector').html($('$selector').html()-1); 60 1000 */ }
